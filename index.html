Welcome to "Cloud Computing"

We provide basic information about cloud computing for students who are new and interested in the field. Topics include:

  • What is cloud computing?
  • Advantages of cloud computing
  • Disadvantages of cloud computing
  • Types of cloud models

What is Cloud Computing?

The term "cloud" refers to the internet. Cloud computing uses remote servers hosted on the internet to store, manage, and process data instead of local computers. This data may include files, documents, images, audio, videos, and more.

Advantages of Cloud Computing

1. Backup & Restore: Easy backup and recovery once data is stored in the cloud.
2. Collaboration: Multiple users can share and work on cloud-stored files easily.
3. Low Maintenance: Reduces the cost of hardware/software maintenance.
4. Mobility: Access data from anywhere using mobile devices.

Disadvantages of Cloud Computing

1. Internet Dependency: Cloud data is only accessible with a reliable internet connection.
2. Security Concerns: Storing sensitive information on third-party servers may involve risks such as data breaches or hacking.

Types of Cloud Models

There are four main types:

1. Public Cloud: Accessible to the public via pay-per-use. Managed by providers (e.g., Amazon EC2, Google App Engine).

2. Private Cloud: Used by a single organization. Offers better control and security. Can be hosted internally or by a third party (e.g., OpenStack).

3. Hybrid Cloud: A mix of public and private clouds, allowing organizations to balance flexibility and security (e.g., AWS + Office 365).

4. Community Cloud: Shared between multiple organizations with common concerns (e.g., healthcare sector). Managed by one or more organizations or a third party.
